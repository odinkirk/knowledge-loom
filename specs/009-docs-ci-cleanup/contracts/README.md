# Contracts: Documentation Cleanup and CI Modernization

This feature introduces no new code interfaces, CLI commands, MCP tools, or API endpoints. All changes are:

1. **Text edits** to root-level markdown files (README.md, CONTRIBUTING.md, Architecture.md)
2. **YAML configuration updates** to GitHub Actions workflows (.github/workflows/*.yml)
3. **One metadata field addition** to Cargo.toml (`rust-version`)

No interface contracts are required.
